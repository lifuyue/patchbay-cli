# Changelog

## 0.1.0 - 2026-06-04

- Initial Issue Finder release for local-first GitHub issue discovery, workspace preparation, and handoff packages for coding agents.
- Includes the JSON tool contract for `issue-finder.scout`, `issue-finder.assess`, `issue-finder.prepare`, and `issue-finder.read_context`.
- Adds cross-platform release installers and GitHub Release assets for macOS, Linux, and Windows.
